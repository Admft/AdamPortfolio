# Adam Portfolio

A React portfolio site built with Create React App, Tailwind CSS, and Framer Motion.

---

## Requirements

- **Node.js** — v18 or later (LTS recommended). [Download](https://nodejs.org)
- **npm** — comes with Node.js (v9+)

Check versions:

```bash
node -v   # e.g. v20.x.x
npm -v    # e.g. 10.x.x
```

If `node` or `npm` is not found, install Node.js from [nodejs.org](https://nodejs.org) or with Homebrew: `brew install node`.

---

## First-time setup

1. **Install Node.js** (if needed)  
   - [nodejs.org](https://nodejs.org) (LTS), or  
   - `brew install node`

2. **Clone and go into the project**
   ```bash
   cd AdamPortfolio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the app**
   ```bash
   npm start
   ```
   Opens [http://localhost:3000](http://localhost:3000).

---

## Available scripts

| Command        | Description |
|----------------|-------------|
| `npm start`    | Run the app in development at [http://localhost:3000](http://localhost:3000) |
| `npm test`     | Run the test runner in watch mode |
| `npm run build`| Production build into the `build` folder |
| `npm run eject`| Eject from Create React App (one-way, not recommended) |

---

## Dependencies

Project dependencies are in `package.json`. After cloning, run `npm install` once to install them. No Python or virtual environment is required.
